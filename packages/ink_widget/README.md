## About

Widget library with ink effect without problems with decoration for child.

## Installation

Add ink widget to your `pubspec.yaml` file:

```yaml
dependencies:
  ink_widget: ^2.0.0
```

You can use both `stable` and `dev` versions of the package listed above in the badges bar.
