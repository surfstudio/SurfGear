## About

Keyboard listener created only on Flutter

## Installation

Add `keyboard_listener` to your `pubspec.yaml` file:

```yaml
dependencies:
  keyboard_listener: ^2.0.0
```

You can use both `stable` and `dev` versions of the package listed above in the badges bar.
