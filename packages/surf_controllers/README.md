## About

Common controller for call dialogs.

## Installation

Add `surf_controllers` to your `pubspec.yaml` file:

```yaml
dependencies:
  surf_controllers: ^1.0.0
```

You can use both `stable` and `dev` versions of the package listed above in the badges bar.
